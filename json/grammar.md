##  grammar
Format of it is `"key" : value`
Use `,` to divide two key-value

###   key

###   value

|--------|--------|----------------------|
| type   | format | notice               |
|--------|--------|----------------------|
| number | `0`    |                      |
|--------|--------|----------------------|
| string | `""`   |                      |
|--------|--------|----------------------|
| logic  | `true` |                      |
|--------|--------|----------------------|
| array  | `[]`   | element is **any**   |
|--------|--------|----------------------|
| object | `{}`   | element is key-value |
|--------|--------|----------------------|
| null   | `null` |                      |
|--------|--------|----------------------|
