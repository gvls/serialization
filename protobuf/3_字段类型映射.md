##  字段类型映射
若传输的时候没有传递，使用默认值

###   常见类型
|----------|---------|---------------------------------|
| proto    | go      | 说明                            |
|----------|---------|---------------------------------|
| double   | float64 |                                 |
|----------|---------|---------------------------------|
| float    | float32 |                                 |
|----------|---------|---------------------------------|
| int32    |         | 变长编码，对负值效率低          |
|----------|---------|---------------------------------|
| uint32   |         | 变长编码                        |
|----------|---------|---------------------------------|
| uint64   |         | 变长编码                        |
|----------|---------|---------------------------------|
| sint32   | int32   | 变长编码，对负值比int32高效     |
|----------|---------|---------------------------------|
| sint64   | int64   | 变长编码，有符号整型比int64高效 |
|----------|---------|---------------------------------|
| fixed32  | uint32  | 若值总是比228大，比uint32高效   |
|----------|---------|---------------------------------|
| fixed64  | uint64  | 若值总是比256大，比uint64高效   |
|----------|---------|---------------------------------|
| sfixed32 | int32   |                                 |
|----------|---------|---------------------------------|
| sfixed64 | int64   |                                 |
|----------|---------|---------------------------------|
| bool     |         |                                 |
|----------|---------|---------------------------------|
| string   |         | UTF-8编码/7-bitASCII编码        |
|----------|---------|---------------------------------|
| bytes    | []byte  |                                 |
|----------|---------|---------------------------------|
